# Contador de Tránsito (Traffic Counter)

Este repositorio se hizo con Angular para poder contar el tránsito de forma rápida. Es un proyecto de prueba satélite a un proyecto más grande de semaforización de una arteria importante de una ciudad.

Puedes ver una demostración en vivo en: [https://counter.jhproyectos.com.ar/](https://counter.jhproyectos.com.ar/)

This project was generated with [Angular CLI](https://github.com/angular/angular-cli).
